## Installation

```bash
npm install @josephavelez77/design-system
```

Install peer dependencies if you haven't already:

```bash
npm install react react-dom \
  @fortawesome/fontawesome-svg-core \
  @fortawesome/free-solid-svg-icons \
  @fortawesome/free-regular-svg-icons \
  @fortawesome/react-fontawesome
```

## Setup

**1. Import the token CSS once at your app root** — this loads all CSS variables and the fonts (DM Sans, DM Serif Display, JetBrains Mono):

```tsx
// main.tsx or _app.tsx
import '@josephavelez77/design-system/styles'
```

**2. Set a theme** — the library defaults to dark mode. Add `data-theme="light"` to your root element for light mode:

```html
<html data-theme="light">
```

Or toggle programmatically:

```ts
document.documentElement.setAttribute('data-theme', 'light')  // light
document.documentElement.removeAttribute('data-theme')         // dark
```

**3. Import and use components:**

```tsx
import { Button, Alert, TextField } from '@josephavelez77/design-system'

export function App() {
  return (
    <>
      <Alert severity="success">Changes saved!</Alert>
      <Button variant="brandPrimary" emphasis="primary">
        Get started
      </Button>
    </>
  )
}
```

## Design tokens

All design decisions are expressed as CSS custom properties. Use them directly in your own styles to stay consistent with the system:

```css
.my-card {
  background: var(--container-color-themeable-primary);
  color: var(--text-color-themeable-primary);
  border: 1px solid var(--border-color-themeable-primary);
  border-radius: var(--border-radius-static-primary);
  padding: var(--container-padding-static-primary);
  gap: var(--container-gap-static-primary);
}
```

Token categories: `border`, `container`, `text`, `icon`, `focus-ring`. Tokens labeled `themeable` respond to the active theme; tokens labeled `static` are the same in both themes.
