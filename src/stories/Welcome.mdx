<h2>Installation</h2>

```bash
npm install @josephavelez77/design-system
```

Install peer dependencies:

```bash
npm install react react-dom \
  @fortawesome/fontawesome-svg-core \
  @fortawesome/free-solid-svg-icons \
  @fortawesome/free-regular-svg-icons \
  @fortawesome/react-fontawesome
```

<hr />

<h2>Setup</h2>

<h3>1. Import token CSS at your app root</h3>
<p>This loads all CSS variables and the fonts: DM Sans, DM Serif Display, and JetBrains Mono.</p>

```tsx
import '@josephavelez77/design-system/styles'
```

<h3>2. Set a theme</h3>
<p>The library defaults to dark mode. Add <code>data-theme="light"</code> to your root element for light mode:</p>

```html
<html data-theme="light">
```

```ts
// Or toggle programmatically
document.documentElement.setAttribute('data-theme', 'light')
document.documentElement.removeAttribute('data-theme') // back to dark
```

<h3>3. Use components</h3>

```tsx
import { Button, Alert, TextField } from '@josephavelez77/design-system'

export function App() {
  return (
    <>
      <Alert severity="success">Changes saved!</Alert>
      <Button variant="brandPrimary">Get started</Button>
    </>
  )
}
```

<hr />

<h2>Design tokens</h2>
<p>All design decisions are CSS custom properties. Use them in your own styles to stay consistent with the system:</p>

```css
.my-card {
  background: var(--container-color-themeable-primary);
  color: var(--text-color-themeable-primary);
  border: 1px solid var(--border-color-themeable-primary);
  border-radius: var(--border-radius-static-primary);
  padding: var(--container-padding-static-primary);
}
```
